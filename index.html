Chat-JS

Real-Time Chat Web Application | Node.js, Express, Socket.IO

By Luca Bocaletto

Test Online Demo

Description

Chat-JS is a self-contained real-time chat web application built with Node.js, Express, and Socket.IO. It features user registration and login with persistent sessions, a real-time chat interface with a persistent message history stored in JSON files, a global notification bell, and a "Clear Chat" button to reset the conversation. Designed for easy deployment on a local network using a friendly domain name.

Features

  • User Registration & Login: Secure registration with persistent sessions.
  • Real-time Chat: Instant messaging using Socket.IO.
  • Persistent Message History: Chat history stored in JSON files.
  • Global Notification Bell: Audible notification for all connected clients.
  • Clear Chat Functionality: Reset the conversation with the "Clear Chat" button.

Project Structure

Installation

Prerequisites

  • Node.js (v12 or later recommended)
  • npm

Steps

  1. Clone the repository: git clone https://github.com/bocaletto-luca/Chat-JS.git
  2. Change into the project directory: cd Chat-JS
  3. Install dependencies: npm install

Usage

  1. Start the server: node server.js
  2. Open your browser and navigate to: http://localhost:3000
  3. Register and log in to start chatting in real time.
